# RA8-GESTI-N-DE-PREGUNTAS-DE-EXAMEN
Aplicación para gestionar un banco de preguntas de examen con base de datos, permitiendo crear, editar, eliminar y consultar preguntas tipo test y desarrollo. Genera exámenes automáticamente según criterios y controla el acceso mediante contraseñas cifradas y registro de actividad.
